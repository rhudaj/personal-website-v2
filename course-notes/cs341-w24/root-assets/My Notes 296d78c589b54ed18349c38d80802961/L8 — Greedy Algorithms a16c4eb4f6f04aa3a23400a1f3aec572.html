Interval Coloring

Idea

We make sure we don‚Äôt overlap with the intervals already covered

  1. We start with the interval that has the earliest start time, ensuring that it doesn't overlap with any previously chosen intervals.
  1. We continue selecting intervals in order of their start times, ensuring that each selected interval doesn't overlap with any previously chosen intervals.

Algorithm

  1. Sort intervals by start time
  1. for each interval: if it does not overlap with the last, give it a different color which does not overlap with any previous color. If no such color exists, give it a new color.
