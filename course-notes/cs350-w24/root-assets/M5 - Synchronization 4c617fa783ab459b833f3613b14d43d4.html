Implications

Avoid state transitions wherever possible (every transition requires expensive bus communications).

So, we can reduce the cache coherence cost by:

  • Avoid false sharing
    • placing data used by different threads in the same cache line
    • multiple diff shared variables, access by different threads, but sitting in memory next to each other â†’ they share the same cache line. If one shared variable is updated frequently, it causes the others to be invalidate too.
    • Donâ€™t have unrelated variables sit next to each other in memory.
  • Align Structures
    • place related data (accessed by same thread) that you often access together (ie: 2 variables) next to each other in memory
    • also, make sure it aligns with the start/end of a cache line
  • Pad data structures
    • Arrays of structures lead to false sharing
    • pad them to align to start/end of cache lines
  • Avoid contending on cache lines
